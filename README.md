# audi-r8
A 3D-printable Audi R8-like model

I made this at the end of my freshman year at TJHSST (2024-25 school year) as a final project for Design and Tech. I had designed the model for fun and chose to modify it for 3D printing and driving as a final project. Unfortunately I no longer have the firmware, but it is quite simple. Just set the servos to the desired speed. I may post an updated version in the future to include steering capabilities and using faster motors. These, however, were the only materials available to me for the assignment.

**Parts required:**
- Standard servo motors *4 (and hardware for mounting)
- **3D printed parts:**
-    Body
-    Door
-    Left tire *2 (TPU)
-   Right tire *2 (TPU
-   Left wheel *2
-   Right wheel *2
-   Arduino UNO R3
-   Jumper wires
-   Breadboard
-   9V battery
-   Laser cut (or 3D printed) bottom plate
-   Hinge
